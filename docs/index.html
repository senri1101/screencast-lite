macOS utility

Shrink huge screen recordings without extra clicks.

screencast-lite uses a LaunchAgent, compresses new Desktop recordings to H.265, and includes a Folder Action compatibility mode for stricter macOS permission setups.

View GitHub Install Guide

How it works

  1. Save a new macOS screen recording (`.mov`) to Desktop.
  2. LaunchAgent receives Desktop file events via `WatchPaths`.
  3. `ffmpeg` creates `_small.mp4` with `libx265` + `crf 28`.
  4. Original `.mov` moves to Trash only after a successful conversion.

One-time setup

Run `setup.command`. It installs Homebrew and `ffmpeg` if needed, then installs and starts the LaunchAgent automatically.

Safe by default

The source file is preserved on conversion failure. Deletion is conditional on success.

Intel + Apple Silicon

The worker script supports both `/usr/local/bin` and `/opt/homebrew/bin` execution paths.

FAQ

Does it support Japanese default recording names?

Yes. It handles the Japanese default prefix internally using Unicode escape sequences.

Can I keep original files?

Yes. Edit the worker shell script and remove the move-to-trash block.

Where can I report issues?

Use the GitHub repository issue tracker.

What if LaunchAgent mode does not detect Desktop files?

Attach `screen_recording_auto_compress.scpt` to Desktop in Folder Actions Setup.
